# Flowtab by Usenov

Flowtab is a modern SaaS landing page demo built as a frontend portfolio project.  
It presents a clean product-style interface for a fictional browser tab management tool.

## Overview

The goal of Flowtab is to showcase modern frontend development through a polished landing page experience: smooth animations, responsive layouts, reusable React components, and clean visual structure.

## Features

- Modern SaaS landing page design
- Responsive layout for desktop, tablet, and mobile
- Smooth reveal animations
- Interactive floating tab elements
- Pricing section with billing toggle
- Mobile navigation menu
- SEO-friendly metadata
- Clean React component structure

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- Vercel
